typedef int(* rbt_walk_callback)(rbt_t *rbt, void *key, size_t klen, void *value, size_t vsize, void *priv)

Callback called for each node when walking the tree.

Parameters
rbtA valid pointer to an initialized rbt_t structure
keyThe key of the node where to store the new value
klenThe size of the key
valueThe new value to store
vsizeThe size of the value
privThe private pointer passed to either rbt_walk() or rbt_walk_sorted()
Returns
1 If the walker can go ahead visiting the next node, 0 if the walker should stop and return -1 if the current node should be removed and the walker can go ahead -2 if the current node should be removed and the walker should stop

Parameters

keyThe key of the node to remove
klenThe size of the key
valueIf not NULL the address of the value hold by the removed node will be stored at the memory pointed by the 'value' argument. If NULL and a free_value_callback is set, the value hold by the removed node will be released using the free_value_callback
vlenIf not NULL the size of the value hold by the removed node will be stored at the memory pointed by the 'vlen' argument
Returns
0 on success; -1 otherwise
