)

Get a copy of the value stored at a specific key.

Parameters
table: A valid pointer to an hashtable_t structure
key: The key to use
klen: The length of the key
dlen: If not NULL, the size of the returned data will be stored at the address pointed by dlen
Returns
The stored value if any, NULL otherwise
Note
The returned value is a copy (memcpy) of the stored value and the caller MUST release it using free() once done
The copy is a simple copy done using memcpy() if the stored value is structured and requires a deep copy, then ht_get_deep_copy() should be used instead of this function
