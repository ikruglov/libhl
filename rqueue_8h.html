Enumeration Type Documentation

enum rqueue_mode_t

The ringbuffer mode.

RQUEUE_MODE_BLOCKING writers will stop if the ringbuffer is full RQUEUE_MODE_OVERWRITE writers will overwrite values when the ringbuffer is full
